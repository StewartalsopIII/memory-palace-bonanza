# The Memory Palace Story

A collaborative story about discovering a mysterious Memory Palace. Each person adds one paragraph.

## How to Contribute

1. Fork this repository
2. Create a branch: `story-[yourname]`
3. Add your paragraph to `story/chapters/[your-paragraph-number].txt`
4. Commit and push to your fork
5. Create a pull request
6. Review someone else's paragraph

## The Story So Far

Read the chapters in order in the `/chapters` folder!

## Rules

- Each paragraph should be 3-5 sentences
- Continue from the previous paragraph
- Keep it PG-rated and fun
- You can add mystery, humor, or unexpected twists
